About

Shiny AnalyzAIRR is a MiAIRR compliant web application for the analysis of bulk Ig/TCR repertoire datasets.

  • It proposes an analytical routine starting with data exploration, leading to in-depth statistical comparisons with the aim of answering defined biological questions
  • It allows the calculation of a set of diversity measures and statistical metrics applicable at any level of granularity
  • It offers different types of data visualization and ready-to-publish graphics

Getting started

Shiny AnalyzAIRR proposes two different data loading processes:

  • Upload a RepSeqExperiment object generated using the AnalyzAIRR package
  • Upload alignment files (i) generated by one of the supported annotation tools, (ii) formatted following the MiAIRR standard guidelines from the AIRR Community, or (iii) following AnalyzAIRR format requirements. Details on the latter format can be found here: .
    Only one chain can be loaded at once.
    It is possible to provide a metadata if users wish to perform inter-group comparative analyses. Sample names matching the alignment files must be in the first column. Any additional columns with relevant information for the analyses can be added thereafter.
Once files are succesfully downloaded, a RepSeqExperiment object is built which can downloaded in a rds format for subsequent usage.
